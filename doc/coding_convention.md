#Coding convention#
## SQL ##
The name of the fields need to be in english and on the singular form.

Primary key: **PK__N**ame
example: **PK__I**D

Foreign key: **FK__N**ame 
example: **FK__U**ser

Name of the compound fields: **m**in**M**aj
example: userAddress
