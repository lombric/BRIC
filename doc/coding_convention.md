#Coding convention#
## SQL ##
The name of the fields need to be in english and on the singular form.

name of the database and the tables: name
example: user

Primary key: **idn**ame
example: **id**name

Foreign key: **fk__n**ame 
example: **fk__u**ser

Name of the compound fields: name_name
example: user__address
